# テトリスゲーム

このアプリケーションは、C++ と SFML を使って作られたシンプルなテトリスのゲームです。  

---

## 🛠 使用ライブラリ

- 'Widows API​'

​- 'Google Apps Script（GAS）で作成したAPI​'


## ⚙️ ビルド方法（Visual Studio）



## ▶️ 実行方法

実行ファイル（.exe）と同じディレクトリに、以下のDLLをコピーしてください：

- `sfml-graphics-d-2.dll`
- `sfml-window-d-2.dll`
- `sfml-system-d-2.dll`

> リリースビルドの場合は `-d` のないバージョンのDLLが必要です。

---

## 🎮 操作方法

- **←→**：ミノの左右移動
- **↑** : ミノの方向変更
